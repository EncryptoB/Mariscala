Cross-Chain Trading Execution System

Laboratorios Uniswap y Across proponen un nuevo estándar para las intenciones de cadena cruzada, estableciendo un marco unificado para los sistemas basados en intenciones para especificar acciones de cadena cruzada. Con esta propuesta, se busca mejorar la interoperabilidad y la experiencia del usuario final en interacciones entre cadenas.

Abstracto: Este estándar permite la implementación de una API estándar para sistemas de ejecución de comercio entre cadenas. Proporciona un struct genérico CrossChainOrder, así como un estándar ISettlementContract para la interfaz de contratos inteligentes.

Motivación: Los sistemas basados en intenciones mejoran la experiencia del usuario al reducir costos, tiempos de espera y tasas de fallas. Este estándar permite a estos sistemas interoperar y compartir infraestructura, beneficiando al usuario final.
